# Roadmap

Swift Package Audit is ready for 1.0 release validation. The JSON/reporting contracts are schema-versioned, config and baseline support are implemented, and the scanner covers basic multi-project workspace layouts.

## Release Validation

- Add more real-world Xcode project fixtures.
- Validate workspace scanning across common project layouts.

## Post-1.0

- License checks.
- CocoaPods support.
- Security advisory checks.
- More package-host-specific release intelligence beyond Git tags.
- Optional fix commands.
