# Roadmap

PackageDoctor is building toward a future 1.0.0 release. A 1.0.0 tag should only be created after the tool has been validated against a broader set of real Xcode projects and the JSON/reporting contracts are considered stable.

## Before 1.0.0

- Add more real-world Xcode project fixtures.
- Validate workspace scanning across common project layouts.
- Decide the stable JSON schema contract.
- Add configuration file support.
- Add baseline support for existing known issues.
- Add GitHub Actions PR comment mode.
- Add optional latest-release and stale-dependency checks.

## Later

- License checks.
- CocoaPods support.
- Carthage support.
- Security advisory checks.
- Optional fix commands.
